Entity == 개체            파이썬의 객체 = 속성 + 기능

--> 객체 - 기능 = 속성 + 기능 - 기능 = 속성 == 개체

--> 개체 = 속성만 가지고 있는 것 ==> table이라는 이름, "entity라고 불리는 테이블"

테이블 간 연결되는 선이 있고, 이를 '관계선'이라고 함.
------------------------------------------------------------------------------
열 = column
행 = row == tuple이라고 부르기도 함

* 데이터를 읽어올 때 단위는 하나의 행 (row) == row == tuple == 객체
